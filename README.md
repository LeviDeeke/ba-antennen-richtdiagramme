Dieses Repository enthält MATLAB-Skripte zur Auswertung und Visualisierung gemessener Antennenrichtdiagramme im Rahmen einer Bachelorarbeit zur Charakterisierung einer Satellitenbodenstation.

Untersucht werden Richtdiagramme einer S-Band-Parabolantenne und einer Yagi-Antenne. Die Skripte erzeugen aus diskreten Messdaten normierte 3D-Richtdiagramme sowie 2D-Azimutschnitte.
