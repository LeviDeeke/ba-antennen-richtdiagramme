Dieses Repository enthält MATLAB-Skripte zur Auswertung und Visualisierung gemessener Antennenrichtdiagramme im Rahmen einer Bachelorarbeit zur Charakterisierung einer Satellitenbodenstation.

Im Fokus stehen die Auswertung und grafische Darstellung von Richtdiagrammen einer S-Band-Parabolantenne sowie einer Yagi-Antenne. Die Skripte dienen dazu, aus diskreten Messdaten normierte 3D-Richtdiagramme und 2D-Polarschnitte zu erzeugen.

Die hier bereitgestellten Dateien enthalten ausschließlich den MATLAB-Code. Die zugrunde liegenden Messdaten sind nicht Bestandteil dieses öffentlichen Repositories.
